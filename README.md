# Hz Cheat

C# .NET WPF приложение для создания читов к одиночным играм.

## Структура проекта

- **Aimbot** - Автоприцел
- **ESP** - Отображение врагов через стены
- **Engine** - Основной движок
- **Memory** - Работа с памятью процесса
- **Offsets** - Смещения в памяти для различных объектов игры
- **Scanning** - Сканирование и поиск смещений
- **UI** - Пользовательский интерфейс

## Компиляция

```bash
dotnet restore
dotnet build --configuration Release
```

## Запуск

Скомпилированный exe находится в `bin/Release/net8.0-windows/`
